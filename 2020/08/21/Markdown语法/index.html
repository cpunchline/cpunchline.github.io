Markdown语法
发表于2020-08-21|更新于2020-10-03|博客
|阅读量:

Markdown 是一种轻量级标记语言，能将文本换成有效的XHTML(或者HTML)文档，它的目标是实现易读易写，成为一种适用于网络的书写语言。本文主要讲述Markdown 基本语法。

标题

要创建标题，请在单词或短语前面添加井号 (#) 。井号的数量代表了标题的级别。

渲染效果如下：

粗体

Markdown 渲染效果
**粗体** 粗体
__粗体__ 粗体
我是**粗体**字 我是粗体字

斜体

Markdown 渲染效果
*粗体* 粗体
_斜体_ 粗体
我是*斜体*字 我是粗体字

粗斜体

Markdown 渲染效果
***粗斜体*** 粗斜体
___粗斜体___ 粗斜体
我是***粗斜体***字 我是**粗斜体**字

块引用

要创建块引用，请在段落前添加一个 > 符号。

有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。

渲染效果如下：

  1. 表项1
  2. 表项2
  3. 表项3
    1. 表项(1)
    2. 表项(2)
  4. 表项4

无序列表

要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。

渲染效果如下：

  • 表项
  • 表项
  • 表项
    • 内表项
    • 内表项
  • 表项

分隔线

要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (---) 或下划线 (___) ，并且不能包含其他内容。为了兼容性，请在分隔线的前后均添加空白行。
